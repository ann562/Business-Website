History of the Internet and Web Technologies
Event Details
Pre-Internet Development 1960s-1980s: Research on packet switching, ARPANET, and early network protocols laid the groundwork for the Internet.
Creation of the World Wide Web 1989-1991: Tim Berners-Lee developed HTML and HTTP, launching the first website in 1991, making the web accessible.
Early Search Engines 1995-1998: Search engines like AltaVista and HotBot helped users navigate the expanding web.
Rise of JavaScript 1995: Brendan Eich created JavaScript, enabling interactivity on web pages, later standardized as ECMAScript.

Web 1.0 Era 1990s: Static websites characterized this era, where content was primarily read-only, with limited user interaction.
Web 2.0 Emergence 2000s: Shift to interactive and social web applications, user-generated content, and collaboration platforms.
Google Search (PageRank) 1998: Larry Page and Sergey Brin launched Google Search with PageRank, revolutionizing search engine algorithms.
REST Architecture 2000: Roy Fielding introduced REST, standardizing web services and enabling stateless communication over HTTP.
Python's Rise 1991: Guido van Rossum released Python, a versatile language that became popular for web development and data science.
Git Development 2005: Linus Torvalds created Git, improving version control for software development and enabling collaboration.
