Web 2.0

Google Search (PageRank)

History:
Larry Page and Sergey Brin, co-founders of Google, launched Google Search in 1998 with the revolutionary PageRank algorithm. This algorithm ranked websites based on the number and quality of links pointing to them, providing more relevant search results than other engines at the time. PageRank laid the foundation for Google's dominance in the search engine market.

  • Who: Larry Page and Sergey Brin
  • Why: To create a more efficient and relevant web search algorithm.
  • When: Launched in 1998

Roy Fielding (REST)

History:
Roy Fielding defined REST (Representational State Transfer) in his 2000 doctoral dissertation as an architectural style for building scalable web services. REST emphasizes stateless communication over HTTP, becoming the most widely adopted standard for web APIs.

  • Who: Roy Fielding
  • Why: To standardize web services architecture.
  • When: Defined in 2000
  • Contribution: REST is a key architectural style for web APIs.

Python (Guido van Rossum)

History:
Python was created by Guido van Rossum in 1991 with a focus on simplicity and readability. It has since become one of the most popular programming languages, known for its versatility in web development, data science, and artificial intelligence.

  • Who: Guido van Rossum
  • Why: To create a simple and readable programming language.
  • When: Released in 1991
  • Contribution: Widely used in web development, data analysis, and AI.

Git (Linus Torvalds)

History:
Git was developed by Linus Torvalds in 2005 to improve version control for software development. Its distributed nature allows developers to collaborate efficiently and manage codebases effectively. Today, Git powers platforms like GitHub and is the most popular version control system.

  • Who: Linus Torvalds
  • Why: To improve version control for software development.
  • When: Released in 2005
  • Contribution: Became the most popular version control system.
