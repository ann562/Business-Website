History of the Internet and Web Technologies

HTML/HTTP Protocol (Tim Berners-Lee)

History:
Tim Berners-Lee, a British computer scientist, invented the World Wide Web in the late 1980s while working at CERN. In 1989, he proposed a system to facilitate information sharing among researchers, leading to the development of HTML (HyperText Markup Language) and HTTP (HyperText Transfer Protocol). HTML enabled the creation and formatting of web documents, while HTTP provided the framework for transmitting those documents over the Internet.

In 1991, Berners-Lee published the first website and made the web accessible to the public. His contributions laid the foundation for the modern Internet, and today, he continues to advocate for an open web focused on standards and privacy.

  • Who: Tim Berners-Lee
  • Why: To create a system for sharing and connecting documents across the internet.
  • When: Developed in 1989-1990
  • Contribution: Invented HTML and HTTP, enabling the first web browsers and servers to communicate.

JavaScript

History:
JavaScript was created in 1995 by Brendan Eich at Netscape Communications. Initially named Mocha, it was later renamed to JavaScript to align with the popularity of Java. JavaScript was introduced to add interactivity to static web pages. After Microsoft developed JScript for Internet Explorer, JavaScript was standardized as ECMAScript by the European Computer Manufacturers Association (ECMA) in 1997.

Over time, JavaScript evolved with major updates like ES5 (2009) and ES6 (2015), becoming essential for web development. Today, it powers dynamic content and interactive applications on both the client and server side through environments like Node.js.

  • Who: Brendan Eich (Netscape Communications)
  • Why: To add interactivity to static web pages.
  • When: Created in 1995
  • Contribution: Became the standard for client-side programming on websites.

CSS (Cascading Style Sheets)

History:
CSS was proposed by Håkon Wium Lie in 1994 to separate content from presentation in web design. The first official CSS1 specification was published in 1996, enabling basic styling for fonts, colors, and layouts. CSS2, released in 1998, introduced new features like absolute positioning and media types. CSS3, launched in 2011, added features such as animations and responsive design.

Today, CSS is a fundamental tool for web design, helping developers create visually appealing and responsive websites.

  • Who: Håkon Wium Lie
  • Why: To separate content from design and provide more control over the presentation of web pages.
  • When: Proposed in 1994
  • Contribution: Essential for styling and layout of web pages.

Perl & PHP (Server-Side Languages)

History:
Perl, created by Larry Wall in 1987, was initially developed for text processing and system administration. It became popular for CGI programming in the early days of the web. PHP, developed by Rasmus Lerdorf in 1995, started as a set of tools for web development and evolved into a powerful language with the release of PHP 3 in 1998.

Perl and PHP played crucial roles in early web development, with Perl excelling at text manipulation and automation, while PHP became widely used for building dynamic websites and content management systems.

  • Who: Perl (Larry Wall), PHP (Rasmus Lerdorf)
  • Why: To enable server-side scripting for dynamic web content.
  • When: Perl (1987), PHP (1995)
  • Contribution: Key role in developing early web applications.

Linus Torvalds (Linux)

  • Who: Linus Torvalds
  • Why: To create a free and open-source operating system.
  • When: Released the first version of Linux in 1991
  • Contribution: Linux became the foundation for many web servers and internet infrastructure.

Early Search Engines (Altavista and HotBot)

  • Who: Altavista (Digital Equipment Corporation), HotBot (Wired Magazine)
  • Why: To make searching the growing web easier.
  • When: Altavista (1995), HotBot (1996)
  • Contribution: Among the first search engines, helping users navigate the early web.
